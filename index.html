Week 1 Vocab

headings:h1-h6 (different sizes, big to small for headings)

p: paragraph seperation

lists:

    ul: dotted point form list
    ol: list is numerical order
    dl: list that does not need to be in order

img: image

figure(img & figcaption): image caption

q: enclosed text in quotation

block quote: a pulled quote from the text

cite: italize things like names of things

em: emphasize text, slightly bolded

strong: very bolded to emphasize a word greatly

b: bolded

i: a part of text in an different mood, italics

Week 2 Vocab

div: block element used to group other elemnts together

span: dividing tag used to mark up a part of a text or part of a document.

a: provides additional information about HTML elements such as an external link with more info(a href=""

p class: use it to class multiple times throughout your html and attach it to different types of elements

p id: only allowed to use to one unique id once in each html page

header: the introductory content, tells the browser where the intro content is

main wrap around the main content of the page and tells the browser this is where the main content is.

section: wraps around related pices of content, usually each section has it's own heading

article: represents a self contained composition in a web page ex. blog post, product card and interactive widgets.

nav: section of the page whose purpose is to provide navigation links. (not used much)

aside: an aside element that's content is only indirectly related to the main content. Often seen as a side bar on a website.

Accessibility HTML

header role="banner": defining an element as the masthead, added to the primary header
