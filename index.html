Intro to HTML

headings: h1-h6 (different sizes, big to small for headings)

p: paragraph seperation

lists:

    ul: dotted point form list
    ol: list is numerical order
    dl: list that does not need to be in order

img: image

figure(img & figcaption): image caption

q: enclosed text in quotation

block quote: a pulled quote from the text

cite: italize things like names of things

em: emphasize text, slightly bolded

strong: very bolded to emphasize a word greatly

b: bolded

i: a part of text in an different mood, italics

Intro to HTML 2

div: block element used to group other elements together

span: dividing tag used to mark up a part of a text or part of a document.

a: provides additional information about HTML elements such as an external link with more info(a href=""

p class: use it to class multiple times throughout your html and attach it to different types of elements

p id: only allowed to use to one unique id once in each html page

header: the introductory content, tells the browser where the intro content is

main wrap around the main content of the page and tells the browser this is where the main content is.

section: wraps around related pices of content, usually each section has it's own heading

article: represents a self contained composition in a web page ex. blog post, product card and interactive widgets.

nav: section of the page whose purpose is to provide navigation links. (not used much)

aside: an aside element that's content is only indirectly related to the main content. Often seen as a side bar on a website.

Accessibility

Landmark Roles

header role="banner": defining an element as the masthead, added to the primary header

nav role="navigation": defining an element as the primary naviagtion, added to the top (nav)

main role="main": defining an element as being the main content of the page, its added to (main)

aside role="complimentary": defining an element that relates to the main content, usually added to an (aside)

footer role="content info"defining an element as information about the content such as copyright and privacy policy, added to (footer)

aria label: a label that will only be announced to people with screen readers, used before a read more link with a description

skip links: placed at the top of the page and offer ways to jump to specific tagged areas of the page

Font Properties

CSS

font-family: The font used throught the website that can be in bold, regular or italics. ex. arial, courier, etc.

font-size: Controls the specific font size for the text

font-weight: Controls the weight of a font. ex. bold, extra bold, thin

font-style Specifies the font style for the text. ex. normal, italic, bold.

line-height: Specifies the height of a line of text. (how tall the text looks)

font-shorthand: Used to shorten the code and specify all individual font properties in one property. shorthand property for:

    font - style
    font - weight
    font - line-height
    font - size
    font - family

Text Decorations

CSS

text-transform: Specifies what to do with the text element such as:

    capitalize
    uppercase
    lowercase
    none

text-decoration: Specifies the decoration used on the text such as:

    line
    color
    style
    thickness

text-align: Specifies the horizontal alignment of text in an element such as:

    center
    left
    right
    justify

text-indent: Specifies the indentation of the first line of text in a text block

text-shadow: Adds shadow to text (add spacing and color)
